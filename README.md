# Le Chiacchiere Cala Millor (Demo)

CaPa Webdesign-Demo für Le Chiacchiere (Carrer na Llambies, 21, 07560 Cala Millor).

Italienisches Restaurant, familiengeführt, ohne eigene Website.

## Subdomain
`lechiacchieredemo.capawebdesign.com`

## Datenquellen
- Adresse, Tel, Hours: Google Maps + Facebook (Ristorantelechiacchiere)
- Beschreibung: Facebook Intro (Spanisch) + Google Reviews
- Empfehlungen (verifiziert in Reviews): Burrata, Pesto Pasta, Carbonara, Spinat-Ravioli, Lachs-Ravioli, Pizza Ziegenkäse-Honig-Zwiebel, hausgemachte Desserts
- Hook: "Pizza Ziegenkäse-Honig" — in fast jeder Bewertung erwähnt
- E-Mail unbekannt → im Sheet vermerkt
